Zed dark theme inspired by [Nord](https://github.com/nordtheme/visual-studio-code) and [Nord Deep](https://github.com/marlosirapuan/vscode-theme-nord-deep).

**Installation**

Copy `yukinord.json` to `~/.config/zed/themes`.
